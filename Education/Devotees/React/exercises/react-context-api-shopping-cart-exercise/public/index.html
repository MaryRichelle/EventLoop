Instructions

Create a basic Shopping Cart logic using Context API

  • Each time the Add to Cart button is clicked on a product, the Total counter in the Header Component gets updated to reflect the total amount.
  • Each time the Add to Cart button is clicked on a product, the Items counter in the Header Component gets updated to reflect the number of products added so far.
  • Each time the Like button is clicked, the Liked counter on the Footer Component must be updated to reflect the number of liked products. The number should not be updated if a product has already been liked.
  • Extras: Make sure that the Like button gets highlighted when the product has been liked and return to normal, when it's not.
